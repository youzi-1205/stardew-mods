# 星露谷物语 Mod 合集（Stardew Valley Mods）

自用星露谷物语 SMAPI mod 合集，主打**减少重复劳动**：自动钓鱼、农场仆人、跨图寻路、箱子代付、库存总览等。

- 运行环境：Stardew Valley **1.6+** / SMAPI **4.0+** / .NET 6
- 界面语言：简体中文
- 玩家直接安装：前往 [Releases](https://github.com/youzi-1205/stardew-mods/releases) 下载 zip，解压到游戏 `Mods` 文件夹即可

## FarmSuite

四个功能合一的综合 mod。

### 1. 制作 / 烹饪直接用箱子里的材料

打开任意制作菜单（背包制作页、厨房、工作台）时，配方的「可制作」判定与扣料**自动计入全世界所有箱子**——含迷你冰箱、厨房冰箱、祝尼魔箱子。不用再搬材料。

### 2. 农场仆人

一个住在农场的帮工 NPC（默认名「仆人」），真人式逐格走动干活：

- **指挥方式**：按 `H`（或右键点他）弹出指令菜单，按附近实际存在的活动态显示选项：
  - 照料附近庄稼（浇水 / 收获 / 施肥 / 补种）
  - 收取机器产物（熔炉、酒桶、鱼饵机等；建筑内的走到门口整栋收）
  - 照料动物（抚摸、补草料、收产物）
  - 碎石头（散石 / 巨石 / 大石块 → 石头矿石）
  - 清木头（树枝 / 树桩 / 原木 / 树苗 → 木头硬木）
  - 除草（杂草 → 纤维）
  - 割草存干草（农场绿草 → 筒仓，按余位限量）
  - 砍附近的树（成年大树，需明确下令）
  - 全农场日常巡一遍
- **早晨自动开工**：每天 6:10 自动巡逻全农场（可关）
- **智能归仓**：收获物与捡到的掉落物优先放进「已有同类物品」的箱子（最近优先），其次最近的空位箱子；建筑内产出优先同屋箱子
- **经验照拿**：收获的耕种经验、砍树伐木经验、碎石采矿经验都记到玩家头上，公式与亲手做完全一致
- **离场继续干**：玩家离开农场后任务以快进模式继续推进
- 种子、肥料消耗自农场箱子；干完活回农舍旁角落待机

### 3. 小皮卡

- 在**罗宾的木匠店**购买（默认 50,000 金，商店列表第一项）
- 骑乘方式同马（右键上下车、进室内自动下车），速度显著快于奔跑
- 按 `U` 打开**车斗货箱**（36 格，跨存档持久）
- 打开世界地图（M）可见皮卡当前位置标记

### 4. 机器一键补货

站在**空闲机器**（熔炉、酒桶、罐头瓶等）2 格内会浮现「一键补货」按钮，点击后自动从你的箱子扣除原料和燃料并开机——**原料和燃料可以分散在不同箱子**，会自动凑齐。

### 主要配置（`config.json`，首次运行后生成）

| 键 | 默认 | 说明 |
|---|---|---|
| `HelperCallKey` | `H` | 呼出仆人指令菜单 |
| `HelperSpeed` | `5` | 仆人移速（玩家奔跑=5） |
| `HelperWorkSpeedMultiplier` | `2.0` | 干活速度倍率 |
| `HelperAutoStartDaily` | `true` | 早晨自动巡逻 |
| `HelperWorkRadius` | `12` | 「附近」扫描半径 |
| `HelperHaulsDrops` | `true` | 捡拾掉落物并归仓 |
| `HelperCutsGrass` | `true` | 启用割草存干草指令 |
| `TruckPrice` / `TruckSpeedBonus` / `TruckCargoKey` | `50000` / `3.0` / `U` | 皮卡相关 |
| `MachineRefillButton` | `true` | 一键补货按钮 |

## FishingAssist

- **自动跟鱼**：小游戏指针自动贴合鱼的位置；**宝箱优先**且保住已有进度
- **自动咬钩**：出现「!」自动起竿（复刻官方自动钩附魔的实现）
- **一键全自动**：在钓过鱼的水边出现「自动钓鱼」按钮，点击后甩竿→咬钩→收鱼→再甩全自动循环；钓到普通鱼自动继续，**有宝箱或背包真的放不下时停下**（同种鱼可堆叠不算满）
- **满力抛竿**：自动钓鱼始终满蓄力（落点更深，鱼质与宝箱概率更好）
- **宝箱增强**（可关）：宝箱出现率小幅提升，开宝箱有概率附加稀有奖励 / 钓鱼武器

配置要点：`AutoCastPower`（抛竿力度）、`TreasureChestChanceMultiplier`（宝箱概率倍率）、`BoostFishingTreasureRarity` 等，详见 `config.json`。

## AutoWalk

- 按 `M` 打开世界地图，**点击任意地名/建筑标签** → 自动跨地图奔跑前往
- 室内目的地（商店等）自动换算为**正门门口**，到门口停下
- 中途**点击鼠标**或按移动键立即停止
- 内置防卡死：被 NPC/动物挡路 0.75 秒即自动绕行；接近出口直接跨图

配置：`OpenMapKey`（默认 `M`）、`RunWhilePathing`（寻路时奔跑）、`StopOnMouseClick` / `StopOnMovementKey`。

## ChestPay

**物品支付类消费不再需要背材料**。打开相关界面时，所需材料自动从你所有箱子「垫付」进背包（原版扣料逻辑自然通过），关闭界面后未消耗的部分自动放回原箱（优先放回有同类物品的箱子）。净效果等同于直接从箱子扣除。

覆盖场景：

- 罗宾 / 法师的**建筑建造**（木头、石头、硬木等，浏览到哪张图纸垫付哪张）
- 罗宾的**房屋升级**（450 木头 / 100 硬木）与**社区升级**（950 木头）
- 克林特的**工具升级**（各级金属锭）
- **修船去姜岛**（200 硬木 + 5 铱锭 + 5 电池组，进船坞自动垫付）
- 沙漠商人等**以物易物商店**（悬停商品即垫付，支持 Shift 批量购买）

## StockOverview

按 `K` 打开**物品总览**面板：

- 汇总背包 + 全世界所有箱子（含迷你冰箱）+ 厨房冰箱 + 祝尼魔箱子 + 皮卡货箱
- 每格显示物品与**总数量**，按类别排序，滚轮翻页
- **悬停**显示该物品的存放分布（哪个地点存了多少）

配置：`OpenKey`（默认 `K`）。

## StardewModAssistant

（可选）Windows 桌面启动器：管理 mod、经由 SMAPI 启动游戏。普通玩家用 Steam 启动选项指向 SMAPI 即可，不需要它。

## 从源码构建

```powershell
# 任一 mod 目录下（需 .NET 6 SDK、已安装游戏与 SMAPI）
dotnet build -c Release
```

借助 [Pathoschild.Stardew.ModBuildConfig](https://www.nuget.org/packages/Pathoschild.Stardew.ModBuildConfig)：

- 自动定位游戏目录并引用游戏程序集
- 编译产物**自动部署**到游戏 `Mods` 文件夹（需游戏未运行，否则 DLL 被占用）
- 同时生成可分发的发布 zip（`bin/Release/net6.0/<名称 版本>.zip`）

## 已知限制

- **多人联机**：仆人与皮卡的核心逻辑运行在房主侧；皮卡购买、货箱按键为房主限定（防止客机花钱买不到东西）。制作取料在多人下未加箱子互斥锁，极端并发可能竞态
- ChestPay 的垫付材料会短暂经过背包（占格子）；背包完全满时会提示清格
- 浣熊许愿（1.6 以物换物）暂未被 ChestPay 覆盖
